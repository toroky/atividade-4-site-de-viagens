📍 ExploreMundo
O ExploreMundo é uma landing page moderna e responsiva desenvolvida para agências de turismo ou exploradores independentes que desejam apresentar destinos incríveis de forma visual e intuitiva.

🚀 Funcionalidades
Design Inspirado em Viagens: Interface limpa com foco em imagens de alta qualidade e tipografia elegante.

Catálogo de Destinos: Seções categorizadas para Praias, Montanhas e Cidades Históricas.

Formulário de Contato: Área dedicada para conversão de clientes com validação visual.

Localização Estratégica: Integração de mapa para exibição da sede da empresa.

Navegação Fluida: Menu fixo (sticky) que facilita o acesso às diferentes seções da página.

🛠️ Tecnologias Utilizadas
HTML5: Estruturação semântica de todo o conteúdo.

CSS3: Estilização personalizada utilizando Variáveis CSS (Custom Properties) para fácil manutenção de cores.

Google Fonts: Uso das famílias Nunito e Playfair Display para um contraste moderno entre textos e títulos.

Font Awesome: Ícones vetoriais para redes sociais e informações de contato.

📂 Estrutura do Projeto
index.html: Página principal contendo o Hero, Destinos Populares, Sobre Nós e Contato.

destinos.html: Página detalhada com a lista completa de destinos e categorias.

mapa.jfif: Recurso visual utilizado para a seção de localização.
